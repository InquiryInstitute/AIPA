# Sustainability Advisor

**Domain**: Environment  
**Agent role**: Compare products, advise on footprint

## Overview

Build an agent that helps users make more sustainable choices—compare products by environmental impact, advise on carbon footprint, or explain regulatory/compliance requirements (e.g., Energy Star, emissions).

## APIs and Data Sources

| Source | Access | Notes |
|--------|--------|-------|
| **EPA APIs** | Free | Air, water, toxics; facility data |
| **Energy Star** | Data files | Product certifications |
| **Emissions data** | Open | Various datasets for carbon, pollutants |

## Agent Components

- **Search**: Find products, facilities, or data by category
- **Compare**: Rank or contrast by sustainability metrics
- **Advise**: Recommend lower-impact options
- **Explain**: Summarize certifications, regulations, or methodology
