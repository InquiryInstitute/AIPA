## Expansion template

Each lecture has:

- **Conceptual Core**: 4–6 prose paragraphs (90-min)
- **===== Key Points**: 6–12 bullets for slides
- **Technical Example**: prose + Key Points
- **Philosophical Reflection**: prose + Key Points
- **Discussion Prompts**: 4–6 prompts
- **Lab Prep**: prose + Key Points, link to labs
