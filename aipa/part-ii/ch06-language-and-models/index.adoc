= Chapter 6: Language and Models

This chapter covers language models: word embeddings, attention and transformers, LLM pretraining and fine-tuning, prompting (zero-shot, few-shot, CoT), knowledge engineering and AI-mediated writing, caching and cost, and the llm tool. The llm tool—embed, generate, prompt, cache—is the agent's language organ. It integrates into student-ai/llm/ and feeds memory (embeddings for RAG), reasoning (CoT), and tools. Labs 1–3 build embeddings, prompting, and the full llm pipeline. By chapter end, the agent can embed, generate, and cache—the linguistic substrate for the rest of the system.

==== Learning Objectives
After this chapter you will be able to:
* Explain word embeddings, attention, and the transformer architecture
* Use prompting strategies (zero-shot, few-shot, chain-of-thought)
* Apply LLM APIs for embedding, generation, and caching
* Build the llm tool (embed, generate, prompt, cache)

==== Course Learning Outcomes (CLOs)

* Explain word embeddings, attention, and the transformer architecture (PLO 1).

==== Chapter Summary
The LLM is the agent's language organ. Embeddings enable similarity; transformers enable generation. Prompting shapes output. The llm tool feeds memory (RAG), reasoning (CoT), and tools. Language models connect to memory (Ch7), reasoning (Ch8), and the agent loop (Ch9).
