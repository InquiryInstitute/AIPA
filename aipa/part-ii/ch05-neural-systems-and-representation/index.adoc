= Chapter 5: Neural Systems and Representation

This chapter covers neural systems: perceptrons and linear boundaries, multilayer networks and backpropagation, training dynamics (vanishing gradients, initialization, batch norm), convolutional networks, regularization (dropout, augmentation), GPU acceleration, and the neural_classifier as a tool. The neural classifier—create_architecture, train, predict, export—integrates into student-ai/ and bridges to Ch6 (LLM). Representation is learned, not programmed. Labs 1–3 build architecture, training with init and regularization, and the end-to-end neural classifier. By chapter end, the agent can train and use neural classifiers for custom tasks.

==== Learning Objectives
After this chapter you will be able to:
* Explain perceptrons, linear separability, and multilayer networks
* Implement backpropagation and understand vanishing gradients, initialization, batch norm
* Use CNNs, dropout, and data augmentation
* Build the neural_classifier tool (create_architecture, train, predict, export)

==== Course Learning Outcomes (CLOs)
By the end of this course, you will be able to:
* Explain perceptrons, linear separability, and multilayer networks (PLO 1).

* Use CNNs, dropout, and data augmentation (PLO 2).

==== Chapter Summary
Neural networks learn representations rather than following programmed rules. Perceptrons, multilayer nets, backprop, and CNNs form the substrate. The neural_classifier tool extends the agent. These representations bridge to language models (Ch6) and embeddings for memory (Ch7).
