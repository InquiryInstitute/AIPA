# Prompt Variants Manual

*For instructor use. Prompt variants are not shown to students before evaluation.*

This manual curates prompt families and example variations by course/chapter. Students receive the family definition and canonical examples; evaluation uses hidden variants along the dimensions in the [Generalization Evaluation Addendum](GENERALIZATION_EVALUATION_ADDENDUM.md).

---

## Usage

- **Canonical prompts** — Share with students as targets.
- **Evaluation variants** — Use in the harness; do not disclose.
- **Variant generators** — Templates for automated variation (e.g., phrasing, domain substitution).
