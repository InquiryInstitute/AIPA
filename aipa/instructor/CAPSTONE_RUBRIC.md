# AIPA Capstone (Ch12) Grading Rubric

Use this rubric for the Ch12 capstone project. Each dimension is scored 1–4; total is the sum (max 20). Full exercise solutions: [EXERCISE_SOLUTIONS.md](EXERCISE_SOLUTIONS.md). See [INSTRUCTOR_MANUAL.md](INSTRUCTOR_MANUAL.md) for the complete instructor guide.

## 1. Problem Definition (max 4)

| Score | Criteria |
|-------|----------|
| 4 | Clear problem, validated with stakeholders; scope is tractable; success criteria defined |
| 3 | Problem clear; some stakeholder validation; scope mostly tractable |
| 2 | Problem vague or too broad; limited validation |
| 1 | Problem ill-defined; no validation |

## 2. Implementation (max 4)

| Score | Criteria |
|-------|----------|
| 4 | Agent integrates 8+ tools; customized for problem; runs reliably; well-documented |
| 3 | Agent integrates 6+ tools; customization present; runs with minor issues |
| 2 | Agent integrates 4+ tools; limited customization; runs with significant issues |
| 1 | Agent incomplete; few tools; does not run |

## 3. Evaluation (max 4)

| Score | Criteria |
|-------|----------|
| 4 | Rigorous evaluation: test set, metrics, baselines, ablation; results documented |
| 3 | Evaluation present; metrics and baselines; some documentation |
| 2 | Evaluation minimal; metrics unclear; no baselines |
| 1 | No evaluation or evaluation invalid |

## 4. Thesis (max 4)

| Score | Criteria |
|-------|----------|
| 4 | Clear structure (problem, approach, implementation, evaluation, limitations); evidence-based; well-written |
| 3 | Structure present; most sections complete; readable |
| 2 | Structure incomplete; gaps in argumentation |
| 1 | Thesis missing or incoherent |

## 5. Presentation (max 4)

| Score | Criteria |
|-------|----------|
| 4 | Clear intro, problem, approach, demo, conclusion; defends design; acknowledges limitations |
| 3 | Most elements present; demo works; some Q&A |
| 2 | Incomplete; demo has issues |
| 1 | Presentation missing or fails to communicate |

## Total

Sum the five scores. Letter grade mapping (adjust for your institution):
- 18–20: A
- 16–17: A-
- 14–15: B+
- 12–13: B
- 10–11: B-
- 8–9: C
- Below 8: Incomplete / needs revision
