## Prerequisites

- Program entry: see [MSc_PROGRAM_SPEC.md](../MSc_PROGRAM_SPEC.md).
- **Chapter-specific:** Ch1–Ch7.

## Course learning outcomes (CLOs)

By the end of this course, students will be able to:

1. Apply propositional and first-order logic (syntax, semantics, resolution) (PLO 2).
2. Use probabilistic reasoning (Bayesian networks, inference, sampling) (PLO 2).
3. Combine LLM chain-of-thought with symbolic verification (PLO 2).
4. Build the reasoning tool (assert, query, prove, infer_probability, chain_of_thought) (PLO 2).
