# AIPA 511 — AI in Institutions

## Credits and contact

- **Credits:** 3
- **Lecture:** 8 sessions × 90 minutes = 12 contact hours
- **Lab:** 12–24 hours (define per institution)

## Prerequisites

- Program entry: see [MSc_PROGRAM_SPEC.md](../MSc_PROGRAM_SPEC.md).
- **Chapter-specific:** Ch1–Ch10.

## Course learning outcomes (CLOs)

By the end of this course, students will be able to:

1. Define governance policies, constraints, and compliance checks (PLO 4).
2. Implement cost control, rate limiting, and responsible AI practices (PLO 4).
3. Conduct A/B testing and bias audit (PLO 3).
4. Build the governance tool (define_policy, check_compliance, simulate, report) (PLO 4).
