## Credits and contact

- **Lab:** 12–24 hours (define per institution)

## Prerequisites

- Program entry: see [MSc_PROGRAM_SPEC.md](../MSc_PROGRAM_SPEC.md).
- **Chapter-specific:** Ch1–Ch3.

## Course learning outcomes (CLOs)

By the end of this course, students will be able to:

1. Explain risk, generalization, bias-variance tradeoff, and overfitting (PLO 1).
2. Implement linear models, gradient descent, regularization (L1/L2), and model selection (PLO 2).
3. Use trees, ensembles (Random Forest, boosting), and evaluation metrics (PLO 3).
4. Build the ml_trainer tool (load_data, train, evaluate, export_model) (PLO 2).
