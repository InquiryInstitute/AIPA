## Course learning outcomes (CLOs)

By the end of this course, students will be able to:

1. Compare pipelines vs. loops and design multi-agent systems (PLO 2).
2. Implement orchestration (tool client) and API integration (PLO 2).
3. Add monitoring, debugging, and scaling to the agent stack (PLO 4).
4. Build the orchestrator integrating llm, memory, reasoning, search, audit, governance (PLO 5).
