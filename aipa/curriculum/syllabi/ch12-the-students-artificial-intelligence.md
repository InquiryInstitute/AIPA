# AIPA 512 — The Future of Intelligence (Capstone)

## Credits and contact

- **Credits:** 3
- **Lecture:** 8 sessions × 90 minutes = 12 contact hours
- **Lab:** 12–24 hours (define per institution)

## Prerequisites

- Program entry: see [MSc_PROGRAM_SPEC.md](../MSc_PROGRAM_SPEC.md).
- **Chapter-specific:** Ch1–Ch11 (all prior courses).

## Course learning outcomes (CLOs)

By the end of this course, students will be able to:

1. Define a capstone problem and validate with stakeholders (PLO 5).
2. Customize the agent for the problem domain (PLO 5).
3. Build an interface (CLI/API) and deploy the system (PLO 5).
4. Evaluate and benchmark the agent; produce the written thesis and presentation (PLO 3, PLO 5).
