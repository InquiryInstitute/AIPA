## Course learning outcomes (CLOs)

By the end of this course, students will be able to:

1. Describe the agent loop (perceive, reason, act) and tool use (PLO 2).
2. Implement tool schemas, registration, and execution (PLO 2).
3. Build the ReAct agent (think → act → observe) with control flow and guardrails (PLO 2).
4. Integrate llm, memory, reasoning, and tools into the tool-calling agent (PLO 5).
