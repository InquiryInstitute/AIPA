## Prerequisites

- Program entry: see [MSc_PROGRAM_SPEC.md](../MSc_PROGRAM_SPEC.md).

## Course learning outcomes (CLOs)

By the end of this course, students will be able to:

1. Explain context limits, external memory, and working memory (PLO 1).
2. Implement vector stores, similarity search, and RAG pipelines (PLO 2).
3. Integrate episodic, semantic, and procedural memory (PLO 2).
4. Build the memory tool (index, search, rag_retrieve, add, update) (PLO 2).
