## Course learning outcomes (CLOs)

By the end of this course, students will be able to:

1. Explain word embeddings, attention, and the transformer architecture (PLO 1).
2. Use prompting strategies (zero-shot, few-shot, chain-of-thought) (PLO 2).
3. Apply LLM APIs for embedding, generation, and caching (PLO 2).
4. Build the llm tool (embed, generate, prompt, cache) (PLO 2).
